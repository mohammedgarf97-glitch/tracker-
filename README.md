## 🚀 خطوات الرفع على GitHub Pages (مجاني)

### الخطوة 1: إنشاء حساب GitHub
1. اذهب إلى **github.com**
2. اضغط **Sign up**
3. أدخل إيميلك وكلمة مرور واسم مستخدم
4. فعّل حسابك من الإيميل

### الخطوة 2: إنشاء مستودع جديد
1. بعد تسجيل الدخول، اضغط **New repository** (الزر الأخضر)
2. اسم المستودع: `student-tracker`
3. اجعله **Public**
4. اضغط **Create repository**

### الخطوة 3: رفع الملفات
1. في صفحة المستودع، اضغط **uploading an existing file**
2. اسحب جميع ملفات هذا المجلد وأفلتها
3. اضغط **Commit changes**

### الخطوة 4: تفعيل GitHub Pages
1. اذهب إلى **Settings** في المستودع
2. من القائمة الجانبية اختر **Pages**
3. في **Source** اختر **main** ثم **/root**
4. اضغط **Save**

### الخطوة 5: الرابط جاهز! 🎉
بعد دقيقتين، رابطك سيكون:
```
https://YOUR_USERNAME.github.io/student-tracker
```
أرسله للطلاب على واتساب!

## ✅ المميزات المتاحة

- [x] لوحة تحكم مع إحصائيات الدفع والغياب
- [x] إدارة الطلاب (إضافة، حذف، تعديل)
- [x] تسجيل الحضور اليومي
- [x] كود QR للحصة مع تجديد تلقائي
- [x] نشر إعلانات موجهة
- [x] مكتبة الموارد التعليمية
- [x] تتبع المدفوعات الشهرية

- [x] ملف شخصي خاص (عزل كامل بين الطلاب)

- [x] مسح كود QR أو إدخاله يدوياً
